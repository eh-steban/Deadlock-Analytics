# Deadlock Analytics

Professional esports analytics platform for competitive Deadlock players and coaches.

## Overview
Deadlock Analytics processes game replay files and provides strategic insights through interactive visualizations, helping players and coaches analyze performance and improve decision-making.

## Features
- Replay file parsing and analysis
- Interactive minimap visualization
- Event-based replay navigation
- Performance metrics and strategic insights

## Tech Stack
- **Backend**: Python (FastAPI), Rust (replay parser)
- **Frontend**: React, TypeScript
- **Database**: PostgreSQL
- **Infrastructure**: Docker, Docker Compose
